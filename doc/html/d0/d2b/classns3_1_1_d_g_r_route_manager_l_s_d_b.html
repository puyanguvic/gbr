Detailed Description

The Link State DataBase (LSDB) of the DGR Route Manager.

Each node in the simulation participating in global routing has a DGRRouter interface. The primary job of this interface is to export Global Router Link State Advertisements (LSAs). These advertisements in turn contain a number of Global Router Link Records that describe the point to point links from the underlying node to other nodes (that will also export their own LSAs.

This class implements a searchable database of LSAs gathered from every router in the simulation.

void ns3::DGRRouteManagerLSDB::Initialize ( )

Set all LSA flags to an initialized state, for SPF computation.

This function walks the database and resets the status flags of all of the contained Link State Advertisements to LSA_SPF_NOT_EXPLORED. This is done prior to each SPF calculation to reset the state of the DGRVertex structures that will reference the LSAs during the calculation.
