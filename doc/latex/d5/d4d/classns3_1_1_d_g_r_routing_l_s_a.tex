\doxysubsubsection{Detailed Description}
a Link State Advertisement (LSA) for a router, used in global routing. 

Roughly equivalent to a global incarnation of the OSPF link state header combined with a list of Link Records. Since it\textquotesingle{}s global, there\textquotesingle{}s no need for age or sequence number. See \textbackslash{}\+RFC\{2328\}, Appendix A.
