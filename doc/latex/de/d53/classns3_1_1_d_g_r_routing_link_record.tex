\doxysubsubsection{Detailed Description}
A single link record for a link state advertisement. 

The Global\+Routing\+Link\+Record is modeled after the OSPF link record field of a Link State Advertisement. Right now we will only see two types of link records corresponding to a stub network and a point-\/to-\/point link (channel).
